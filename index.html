JS & jQuery

JavaScript and jQuery are scripting languages used to add interactivity and dynamic behavior to web pages. JavaScript is the more fundamental language, providing a wide range of functionalities for manipulating the Document Object Model (DOM), handling user events, and making network requests. jQuery is a popular JavaScript library that simplifies and streamlines common DOM manipulation tasks, event handling, and animations.

GIThub

GitHub is a web-based hosting service for version control using Git. It is widely used for software development, documentation, and collaboration.

GitHub allows developers to store, manage, and track code changes, as well as collaborate with others on projects. It provides a central repository for code, allowing developers to fork and pull changes from others.

React & React
Native

React and React Native are JavaScript libraries for building user interfaces (UIs). React is primarily used for web development, while React Native is designed for mobile app development.

Both libraries employ the component-based approach, breaking down UIs into reusable and manageable components.

HTML 5&4

HTML 5 is the latest standard for the HyperText Markup Language (HTML), the foundational language for creating web pages. It introduces new features that enhance the accessibility, responsiveness, and interactivity of web content.

CSS 3

CSS3 is the latest version of the Cascading Style Sheets (CSS) language, responsible for styling and presentation of web pages.
